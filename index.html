Aprendendo a aprender

Finalmente meu primeiro projeto aqui na Trybe! Rodeado de profissionais incríveis, colegas de turma sensacionais. Em um ambiente super colaborativo, mas que valoriza muito a autonomia… sem dúvida foram dias de muito aprendizado.

Desde os Welcome Days, foram três blocos dentro do módulo de Fundamentos do Desenvolvimento Web. Vou contar um pouquinho sobre minha experiência e o que venho aprendendo desde então. #VQV

Bloco 01: Unix e Bash

Neste bloco aprendi conceitos importantes sobre o sistema operacional Unix. Criado por Ken Thompson, Dennis Ritchie e sua equipe em 1973 como um sistema multiúso e multitarefas.

Também aprendi sobre Shell, interpretador de linha de comando que facilitou muito o uso dos computadores proporcionando uma interface mais amigável fazendo uma ponte entre o Kernel “raiz do sistema operacional” e o usuário.

De forma mais prática aprendi sobre diretórios. Como criar e manipular diretórios e arquivos a partir de comandos usando o Bash “Unix shell”.

Também aprendi sobre busca de arquivos através do comando “find”, concatenação de comandos utilizando o “|”, processos e permissões em arquivos e mais.

Bloco 02: Git, GitHub e Internet

Neste bloco aprendi sobre sistemas de versionamento local e remoto. Algo importantíssimo para qualquer desenvolvedor, proporcionando um trabalho colaborativo muito mais ágil e seguro.

Além disso aprendi na prática como trabalhar com repositórios no Git e Github. Criação e clonagem de repositórios, controle de arquivos através de commits, push e pull requests etc.

Neste bloco também aprendi um pouco sobre o funcionamento da internet, modelos client servidor, protocolos de segurança, requisições etc. Informações importantes que servirão de base durante os estudos sobre aplicações web.

Bloco 03: Introdução à HTML e CSS

Neste bloco aprendi sobre como estruturar documentos HTML de forma semântica, utilizando tags que trazem mais significado a nossas páginas, além de facilitar a leitura das páginas nos readers.

Além disso, aprendi sobre CSS “Cascading Style Sheets”. Através do css é possível estilizar elementos de páginas HTML alterando comportamento, posicionamento, esquema de cores e muito mais.
